La función map

Función map/2

  • Dada una lista xs y una función f, aplica la función f sobre cada elemento de la lista xs.
  • Devuelve otra lista con los resultados.

Definición

  • La función map se encuentra definida en el módulo Enum.
  • Una versión específica para listas puede definirse del siguiente modo:
def map([], _), do: []
def map([x | xs], f) do
  [f.(x) | map(xs, f)]
end

Ejemplos

> map([1, 4, 5, 3], fn x -> x + 1 end)
[2, 5, 6, 4]

> map([1, 4, 5, 3], &(&1 * 2))
[2, 8, 10, 6]

> map([:a, :b, :c], &to_string/1)
["a", "b", "c"]

> map([{1, :a}, {2, :x}, {3, :d}, {4, :g}], fn {x, _} -> x end)
[1, 2, 3, 4]

La función filter

Función filter/2

  • Dada una lista xs y una función f, construye otra lista con aquellos elementos x de xs tales que f.(x) devuelve un valor cierto.

    • (cierto = distinto de false o nil)

Definición

  • La función filter se encuentra definida en el módulo Enum.
  • Una versión específica para listas puede definirse del siguiente modo:
def filter([], _), do: []
def filter([x | xs], f) do
  if f.(x) do
    [x | filter(xs, f)]
  else
    filter(xs, f)
  end
end

Ejemplos

> filter([1, 2, 3, 4, 5], &(rem(&1, 2) == 0))
[2, 4]

> filter(["Aarón", "Alberto", "Pilar", "Ainara", "Silvia"], 
         &(String.starts_with?(&1, "A"))))
["Aarón", "Alberto", "Ainara"]

> filter([[2, 3], [1, 1, 1], [3, 4], [1], []], 
            fn [] -> false
               [x | _] -> x >= 2
            end)
[[2, 3], [3, 4]]

La función reduce

Función reduce/3

  • Recibe una lista xs = [x1, …, xn], un valor inicial ini y una función f de aridad 2. Esta última función recibe un elemento de xs y un acumulador.

    • Comienza invocando f(x1, ini). El resultado pasa a ser el acumulador ac1. Después invoca f(x2, ac1) obteniendo otro acumulador ac2, y así sucesivamente.

  • Cada valor devuelto por f es el acumulador que se pasa a la siguiente invocación de f.

  • Devuelve el valor del acumulador tras aplicar f al último elemento de la lista.

Función reduce/3

Definición

  • La función reduce/3 se encuentra definida en el módulo Enum.
  • Una versión específica para listas puede definirse del siguiente modo:
def reduce([], ac, _), do: ac
def reduce([x | xs], ac, f) do
  reduce(xs, f.(x, ac), f)
end

Ejemplos

> reduce([2, 4, 1, 8], 0, &(&1 + &2))
15

> reduce([2, 4, 1, 8], 1, &(&1 * &2))
64

> reduce([:a, :b, :c, :d], 0, fn _, ac -> ac + 1 end)
4

> reduce([:a, :b, :c, :d], [], fn x, ac -> [x | ac] end)
[:d, :c, :b, :a]

Variante: reduce/2

  • Funciona igual que reduce/3, pero no recibe el valor inicial del acumulador.
  • Supone que el valor inicial es el primer elemento de xs, y comienza a aplicar a la función f a partir del segundo elemento.

  • Puede definirse reduce/2 a partir de reduce/3:

    def reduce([x | xs], f) do
  reduce(xs, x, f)
end

  • La función reduce/2 no puede aplicarse a listas vacías.

  • Ejemplos:

    > reduce([2, 4, 1, 8], &(&1 + &2))
15

> reduce([2, 4, 1, 8], &(&1 * &2))
64

> reduce([1, 5, 7, 2], &max/2)
7

Variante: foldr/3

  • Definida en el módulo List.

  • Funciona igual que reduce/3, pero recorre la lista de derecha a izquierda.

La función flat_map

Función flat_map/2

  • Al igual que map/2, recibe una lista xs y una función f.
  • Aplica f a cada elemento de xs, pero esta vez supone que f devuelve una lista.
  • flat_map devuelve el resultado de concatenar todas las listas devueltas por f.

Definición

  • La función flat_map/2 se encuentra definida en el módulo Enum.
  • Una versión específica para listas puede definirse del siguiente modo:
def flat_map([], _), do: []
def flat_map([x | xs], f) do
  f.(x) ++ flat_map(xs, f)
end

Ejemplos

> flat_map([1, 5, 7, 8], fn x -> [x, x] end)
[1, 1, 5, 5, 7, 7, 8, 8]

> flat_map([[1, 4], [1, 5], [7, 8, 6]], fn x -> x end)
[1, 4, 1, 5, 7, 8, 6]

La función zip_with

Función zip_with/3

  • Similar a map, pero fusiona dos listas en una sola.

  • Recibe dos listas (xs e ys) y una función f de aridad 2.

    xs = [x1, x2, …, xn]    ys = [y1, y2, …, yn]

  • Devuelve una lista con los resultados de aplicar f sobre los elementos de xs e ys:

    zip_with(xs, ys, f) =
      [f(x1, y1), f(x2, y2), …, f(xn, yn)]

  • Si las listas no tienen la misma longitud, el resultado tiene la longitud de la lista más corta.

Definición

  • La función zip_with/2 se encuentra definida en el módulo Enum.
  • Una versión específica para listas puede definirse del siguiente modo:
def zip_with([], _, _), do: []
def zip_with(_, [], _), do: []
def zip_with([x | xs], [y | ys], f) do
  [f.(x, y) | zip_with(xs, ys, f)]    
end

Ejemplos

> zip_with([1, 6, 9], [3, 2, 5], &(&1 + &2))
[4, 8, 14]

> zip_with([10, 4, 3], [:a, :b], &{&1, &2})    Hace lo mismo que `zip/2` 
[{10, :a}, {4, :b}]

Más información: https://hexdocs.pm/elixir/Enum.html#zip/2
